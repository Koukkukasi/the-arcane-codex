## Quick Start Guide

### Path 1: Just Want to See It? (2 minutes)
1. Open: `INVENTORY_OVERLAY_COMPLETE.html` in your browser
2. Press 'I' key to toggle inventory
3. Click items to see tooltips

### Path 2: Ready to Integrate? (5-10 minutes)
1. Read: `COPY_TO_MAIN_HTML.txt`
2. Copy CSS section to your `<style>` tag
3. Copy HTML section before `</body>`
4. Copy JavaScript in `<script>` section
5. Test with 'I' key

### Path 3: Need Full Details? (15+ minutes)
1. Read: `INVENTORY_OVERLAY_README.md`
2. Review: `INVENTORY_VISUAL_GUIDE.txt`
3. Follow: `COPY_TO_MAIN_HTML.txt`
4. Customize using examples
5. Test thoroughly
